Formatação

Negrito - Para adicionar negrito, usamos a tag "b".

Itálico - Para adicionar itálico, usamos a tag "i".

Destacado - Para adicionar um destaque, usamos a tag "mark".

Tachado - Para adicionar um risco, usamos a tag "del".

Sobrescrito - Para adicionar um sobrescrito, usamos a tag "sup".

Subescrito - Para adicionar um subescrito, usamos a tag "sub".

Listas

* Lista não Ordenada

Para declarar uma lista não ordenada, usamos a tag "ul" e para cada elemento da lista a tag "li".

  • Café
  • bolacha
  • Aveia
  • goiaba

* Lista Ordenada

Para declarar uma lista Ordenada, usamos a tag "ol" e para cada elemento da lista, a tag "li".

  1. Café
  2. bolacha
  3. Aveia
  4. goiaba

Para as listas Ordenadas, podemos colocar diferentes tipos e enumerações.

Enumerações: Arábicas, Alfabéticas (maiusculas e minusculas) e Romanas. Definidas pelo "type" após declarar o "ol"

Seletores

É uma ferramenta que permite selecionar determinados elementos HTML para estilizar via CSS, sem que seja necessário mudar todo o HTML.

* Classes

Esse seletor permite alterar a estilização de determinado grupo de elementos que faça parte da classe.

Para criar uma classe, declaramos 'class="_nome_da_classe_"' na tag desejada.

* Identificadores

Esse seletor permite alterar a estilização de um elemento em especifico desde que seja especificado qual.

Para criar um identificador, declaramos 'id="_nome_do_identificador_"' na tag desejada.

Atenção: o elemento identificado deve ser único
